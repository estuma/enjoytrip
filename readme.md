---

## 🔎 주요 기능

1. **여행 경로 설정**
    
   - 가고싶은 여행지를 선택하고 **방문 일자**와 **방문 순서**를 설정할 수 있습니다.
   - **보관함** 을 통해 여행 계획을 좀 더 자유롭게 세울 수 있습니다.
   - 여행지를 검색하고 가고싶은 여행지를 **찜** 할 수 있습니다.
   
    
2. **사진 공유**   

    - 다녀온 여행지의 사진을 공유할 수 있습니다.
